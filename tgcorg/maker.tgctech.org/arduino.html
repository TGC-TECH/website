Arduino 101

This roughly 5 to 6 hour course can be broken up into upto 5 segments, The course will explain the basics of arduino hardware and C++ software in the Arduino IDE including:
-Basic C++ programming syntax and commands for arduino
-Using digital inputs (buttons and switches)
-Using digital outputs (LEDs, buzzers, etc)
-Using analog inputs (light sensors and temp sensors)
-Using complex bus outputs such as i2c to drive an OLED display module
Price $150 Includes an arduino nano, breadboard, 50 LEDs, a buzzer, a light sensor, a temp sensor, 10 buttons, 2 switches, and an OLED display module
