Intro to Raspberry Pi

This 5 to 6 hour course explains the basic of Raspberry Pi hardware as well as Linux (bash) and Python software. The course covers
-How to use imaging and disk management software such as etcher to create bootable media for the raspberry pi
-How to use VLC and X server to manage your raspberry pi
-how to use SSH to manage your raspberry pi through terminal
-How to use digital inputs to control digital outputs with Python
--How to use digital inputs to control the linux operating system with python and bash
